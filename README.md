# NetBox Facilities

NetBox Facilities is a [NetBox](https://github.com/netbox-community/netbox) plugin enables modeling architectural, structural, MEP
(mechnical, electrical, and plumbing) and security facilities that make up the gray space of a data center and support the ITE (IT
equipment) in the white space.

The objective is to build out a comprehensive data model that incorporates the [ANSI/TIA-942 Telecommunications Infrastructure Standard for Data Centers](https://tiaonline.org/products-and-services/tia942certification/ansi-tia-942-standard/) and
[ANSI/BICSI 002 Data Center Design and Implementation Best Practices](https://www.bicsi.org/standards/available-standards-store/single-purchase/ansi-bicsi-002-the-standard-for-data-center-design)
standard and integrate defined ratings/availability classes and standard-compliance designs at a DCIM level.
