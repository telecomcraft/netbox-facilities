# netbox-facilities

Netbox Facilities enables modeling architectural, structural, MEP (mechnical, electrical, and plumbing) and security facilities that
make up the gray space of a data center and support the ITE (IT equipment) in the white space. The objective is to
build out a comprehensive data model that incorporates the BICSI 002 Data Center Design and Implementation Best
Practices standard and integrate standard-compliance designs at a DCIM level.
